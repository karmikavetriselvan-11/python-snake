# Snake Game

A classic grid-aligned Snake Game built using Python's standard `turtle` library. 

This version includes gameplay improvements such as perfect grid alignment for food, speed scaling as the score increases, a high score tracker, and graceful closure handling.

## Features

- **Grid Alignment:** The food aligns perfectly with the 20-pixel grid, ensuring clean collisions.
- **High Score Tracker:** Keeps track of your best run during the session.
- **Dynamic Difficulty:** The game speeds up slightly every time the snake eats food.
- **Graceful Termination:** Closing the window exits the program cleanly without throwing errors.

## Controls

Use the keyboard arrow keys to guide the snake:
- **Up Arrow:** Move Up
- **Down Arrow:** Move Down
- **Left Arrow:** Move Left
- **Right Arrow:** Move Right

## Getting Started

### Prerequisites

You need **Python 3** installed on your system. Python's default installation comes with `tkinter` and `turtle` pre-installed on Windows.

### Installation & Run

1. Clone or download this repository.
2. Open a terminal (PowerShell, Command Prompt, or bash) in the project directory.
3. Run the following command:
   ```bash
   python snake.py
   ```
